# OAuth Note

This project does not use a normal public consumer OAuth setup like WHOOP, Strava or Fitbit.

Garmin's official Health API is partner-approved. Garmin MCP Unofficial uses personal Garmin Connect token mode through a local helper. See [Auth](./auth.md).

For this personal local mode, users do not need a Garmin developer app, client ID, client secret, redirect URI or scopes.
